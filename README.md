Application de Veille Technologique IA
Cette application Streamlit a été créée dans le cadre d'un challenge et présente une veille technologique sur les dernières avancées en intelligence artificielle.

Fonctionnalités
Authentification utilisateur : Système de connexion sécurisé basé sur un fichier CSV
Navigation intuitive : Menu latéral pour naviguer entre les différentes sections
Veille IA : Présentation des trois dernières innovations majeures en IA, chacune avec une illustration et un descriptif
Structure de l'application
Page d'accueil : Introduction générale à l'application
Page Veille IA : Présentation des trois innovations récentes en intelligence artificielle
Système d'authentification : Gestion des connexions utilisateurs avec différents rôles
Identifiants de connexion
Pour tester l'application, vous pouvez utiliser les identifiants suivants :

Utilisateur standard :
Username: utilisateur
Password: utilisateurMDP
Administrateur :
Username: root
Password: rootMDP
Déploiement
Cette application est prête à être déployée sur Streamlit Cloud. Il suffit de :

Créer un compte sur Streamlit Cloud
Déposer les fichiers sur un repository GitHub
Déployer l'application depuis l'interface Streamlit Cloud
Fichiers inclus
app.py : Code principal de l'application
users.csv : Base de données des utilisateurs
requirements.txt : Dépendances nécessaires pour le déploiement
